# Git-GitHub---Partie-2-Activit-
Activité du cours "Gérez votre code avec Git et GitHub

Pour cette activité, je vais utiliser le code html et css de l'activité d'un autre cours.

Construire son cv (cours "créer votre site web avec HTML5 et CSS3")

Le but de l'activité était de créer son cv en html5 et css3 et qu'il soit responsive.
C'est à dire qu'il s'adapte à la largueur de l'écran (pour les mobiles notamment).

Il y a donc du HTML, du CSS, des photos et des police d'écriture.
